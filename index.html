Pirates Hang

Pirates Hang is a browser based pirate themed hangman game. The user is given a random word and a random number of wrong guesses available. Pirates Hang is all front end based and uses vanilla javascript for all dom manipulation and game play.
Skills used: HTML, CSS, and Javascript.

You can demo Pirates Hang here

Avengers RPG

Avengers RPG presents users with a chance to play as one of 4 avengers. They must then carefully select the order of who to battle in an attempt to beat the other 3 avengers. Avengers RPG utilizes bootstrap for it's layout and jQuery for dom manipulation.
Skills used: HTML, CSS, Bootstrap, javascript, and jQuery.

You can demo Avengers RPG here

Eatinerary

Eatinerary was a group project I built alongside David Quinlan and Herbert Gutierrez. We built Eatinerary with an objective of providing one place in which a person can search restaurants across the nation, see their ratings and contact information, and add restaurants onto their own unique to-do-list with an option to receive an email reminder on the date they set. In the construction of Eatinerary I was primarily responsible for the routing, AJAX calls, and setting up a mailing service on our server.

Technologies used: Javascript, Node.js, Express, mySQL, Sequelize, Node Mailer, jQuery, AJAX, APIs, Materialize, HTML, and CSS.

You can demo Eatinerary here

Coders Connect

Coders Connect was my first experience building both the front and back end, and it was a blast. Coders Connect takes you through a survey asking you about your programming preferences, and after the survey is complete it pairs you with another user who answered the most similar to your self. I used Node.js along with Express to provide the http routing and comparison function on the back end.

Skills used: Node.js, Express, Javascript, jQuery, Bootstrap, HTML, and CSS.

You can demo Coders Connect here

Eat-Da-Burger

In Eat-Da-Burger I leveraged Node.js, Express, mySQL, and handlebars.js to create a server rendered site. The actual user functionality of Eat-Da-Burger is very simple: Give the user a list of undevoured burgers, a list of eaten burgers, and an option to add a new burger.

Skills used: Node.js, Express, Javascript, mySQL, Bootstrap, Handlebar templates, and CSS.

LIRI

LIRI is a language intepretation and recognition interface. LIRI can retrieve your last 20 tweets, search any song to retrieve the stats along with a spotify link for the top twenty results, and can give you a movie description for any movie you search. LIRI works in the command line and utilizes the twitter and spofity npm packages.

Skills used: Node.js, Javascript.
